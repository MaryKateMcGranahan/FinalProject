Picture Break!

Decoding the Vignere cipher was a lot of letters. Sending long strings of jumbled up letters is inconvient, and not very secure.
Both the Caesar and Vignere cipher's can be solved by anyone who intercepts the message, without even having the key. There is only a certain number of combinations it can be.
What if you could send messages using pictures instead?
These adorable puppies are actually secret agents. Their pictures contain a message in their binary code.

Every pixel in a picture corresponds to a hexadecimal number. The code behind the first puppy looks like this:
Hexadecimal numbers are base 16. We usually count in base 10: 1, 2, 3... Binary is base 2, only 1's and 0's. These hexadecimal numbers can be converted to binary. The message is stored in the binary code.
